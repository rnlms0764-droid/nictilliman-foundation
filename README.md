# The NicTilliman Foundation Inc Website

A professional, mobile-friendly website for The NicTilliman Foundation Inc, a 501(c)(3) charitable organization serving High Springs, Florida.

## About

- **Organization:** The NicTilliman Foundation Inc
- **Location:** High Springs, FL
- **EIN:** 39-3751590
- **Status:** 501(c)(3) Public Charity

## Website Features

- ✅ 6 professionally designed pages
- ✅ Mobile-responsive design
- ✅ SEO optimized
- ✅ Fast loading (pure HTML/CSS/JS)
- ✅ Donation CTAs
- ✅ Contact form
- ✅ FAQ section
- ✅ Social media integration

## Pages

1. **Home** - Welcome, mission preview, impact stats
2. **About** - Foundation story, founder info, values
3. **Mission** - Detailed mission, vision, focus areas
4. **Events** - Community programs, volunteer opportunities
5. **Donate** - Ways to give, FAQ, support options
6. **Contact** - Contact form, location, social links

## Technology

- HTML5
- CSS3 (custom, no frameworks)
- Vanilla JavaScript
- Google Fonts (Inter + Playfair Display)
